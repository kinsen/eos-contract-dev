# README

Scaf for develop EOS contract use VSCode

## HOW TO USE?

- create project
    ```bash
    $ git clone 
    ```

- update `CMakeLists.txt` set **ProjectName** to your project name.
- create a dir under `contracts`

## How to gen abi?

- select the contract folder, such like `hello`.
- ⇧⌘P
- Run Task
- Generate ABI
